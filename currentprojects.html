Current Projects


PetBook

This project was born to address the challenges we faced with lost pets and trying to locate and return them to their owners. We realized quickly that although apps like Nextdoor existed, there wasn't anything out there that focused on pets and helped make this process easy.

This project is the UI/UX design and development for the app. It's a group project with myself and another UI/UX designer working on design and front end and a developer for the back end.




TriUXPA

The Triangle UXPA was founded in 2006 and is the Triangle chapter of the User Experience Professionals Association. TriUXPA is focused on building and supporting the local community of practitioners who focus on crafting great user experiences.

As a member of TriUXPA I noticed that although their website contained all the information it was not structured well, was cluttered and it did not have a good user experience. So this project is a redesign and code prototype of the TriUXPA website with myself and another UI/UX designer working on design and development.




Tripster

This is a travel app that centers around travel journaling.

This project is the UI/UX design and development for the app. I am working on the design and front end along with a developer on the back end.




School Navigator

Durham School Navigator is web application tool to help parents understand the different public schools in Durham County and make informed schooling decisions. Durham School Navigator provides a common and easy-to-use platform for all public schools to share information about themselves with parents.

This is a project being developed as part of the Code for Durham civic hacking initiative. For this project I am part of the design team working on UI and UX design.




Kids Coding
